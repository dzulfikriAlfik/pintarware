# Pintarware — Platform Edukasi

Website Pintarware untuk edukasi programming: kursus online, blog, produk digital, dan mentoring.

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS 4
- Framer Motion
- React Router v7

## Menjalankan

```bash
npm install
npm run dev
```

Buka http://localhost:5173

## Struktur Halaman

- **/** — Home
- **/courses** — Kursus online (gratis & berbayar)
- **/blog** — Blog & artikel
- **/products** — Produk digital (template, starter kit)
- **/mentoring** — Mentoring 1-on-1 (rate per jam)

## Design System

- **Primary:** Teal (#0F766E) — trust, learning
- **Accent:** Amber (#D97706) — CTA, achievement  
- **Typography:** DM Sans (body), Fraunces (display)
