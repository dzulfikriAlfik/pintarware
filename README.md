# Pintarware — Education Platform

Pintarware website for programming education: online courses, blog, digital products, and mentoring.

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS 4
- Framer Motion
- React Router v7

## Running

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Page Structure

- **/** — Home
- **/courses** — Online courses (free & paid)
- **/blog** — Blog & articles
- **/products** — Digital products (templates, starter kits)
- **/mentoring** — 1-on-1 mentoring (hourly rate)

## Design System

- **Primary:** Teal (#0F766E) — trust, learning
- **Accent:** Amber (#D97706) — CTA, achievement  
- **Typography:** DM Sans (body), Fraunces (display)
